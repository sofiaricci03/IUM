## Progetto Interfaccia Uomo-Macchina

### come riprodurre in localhost:

1. scaricare repo
2. download .NET SDK: 
https://dotnet.microsoft.com/en-us/download
3. dalla root spostarsi in Template.Web e compilare:
   cd /Unibo-Template/src/Template.Web
   dotnet build

4.avviare l'app
dotnet run (se tutto è stato compilato correttamente la web app sarà disponibile su http://localhost:5178
