### come riprodurre in localhost:

1. scaricare repo

2. download .NET SDK: 
https://dotnet.microsoft.com/en-us/download
``` 
npm install 
``` 
3. dalla root spostarsi in Template.Web e compilare:

 ``` 
   cd /Unibo-Template/src/Template.Web
   
   dotnet build
   
   dotnet run
``` 

4. navigare al http://localhost:5178

5. login dipendente:
``` 
dipendente1@azienda.it
password1
dipendente2@azienda.it
password2
dipendente3@azienda.it
password3
```
